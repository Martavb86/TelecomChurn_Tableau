# Telecom Churn Analysis Dashboard (Tableau)

## Description
This is an interactive **Tableau Public dashboard** analyzing customer churn in a telecommunications company.  
It displays key KPIs, charts, and allows exploration by gender, contract type, and payment method.

You can **view the dashboard online** here: [Telecom Churn Tableau Dashboard]  https://public.tableau.com/views/TelecomChurn_17679699098730/TelecomChurnDashboard?:language=es-ES&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

---

## Dataset
The dataset `telecom_churn` includes:

- `customerID`, `Gender`, `SeniorCitizen`, `Partner`, `Dependents`  
- `Tenure`, `Contract`, `PaymentMethod`, `MonthlyCharges`, `TotalCharges`, `Churn`  

---

## Key Visualizations

### 1. Main KPIs
- Total Customers  
- Clients Churn  
- Churn Rate (%)

### 2. Interactive Charts
- Churn by Contract  
- Revenue by Payment Method  
- Tenure Distribution

### 3. Filters
- Gender  
- Contract  
- Payment Method

> All visuals update dynamically when filters are applied.

---

## Technologies Used
- **Tableau Public**  
- **Calculated fields** for KPIs  
- Interactive dashboards and filters
